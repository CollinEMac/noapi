
## Examples

### 1. How to run

Take 01-api-definition as an example:

```sh
cd /path/to/noapi
cd ./examples/01-api-definition
```

Read define.js:
```sh
open ./api/define.js
```

Run:
```sh
node .
```

Test:
```sh
npm test
```
