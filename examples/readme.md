
## Examples

### 1. How To Run A Single Example

Take example 01 as an example:

```sh
cd /path/to/noapi/examples/01-api-definition-by-array
```

Read define.js:
```sh
open ./api/define.js
```

Run:
```sh
node .
```

Test:
```sh
npm test
```



### 2. How To Run All Examples


```sh
cd /path/to/noapi/examples
sh ./test.sh
```
